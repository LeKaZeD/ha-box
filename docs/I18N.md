# Support multilingue (i18n) - HA Box

## Vue d'ensemble

Le support multilingue permet de traduire :
- **Interface de configuration** : Labels et descriptions dans Home Assistant
- **Messages utilisateur** : Textes affichés sur l'écran E-Paper, logs

## Fonctionnement

### Fichiers de traduction

Les traductions sont stockées dans `translations/{langue}.yaml` (ex: `fr.yaml`, `en.yaml`).

Home Assistant charge automatiquement les traductions selon la langue de l'interface utilisateur pour les labels/descriptions dans `config.yaml`.

### Utilisation dans le code Python

Un module `i18n.py` sera créé pour charger les fichiers YAML et fournir une API simple :
- Détection automatique de la langue (variable d'environnement ou défaut EN)
- Chargement des traductions depuis les fichiers YAML
- Méthode `get(key)` pour récupérer une traduction

### Langues prévues

- **Français (fr)** : Prioritaire
- **Anglais (en)** : Prioritaire
- Autres langues : Optionnel (DE, ES, etc.)
